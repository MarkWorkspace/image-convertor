# Программа для конвертации изображений в формат .webp

## Компиляция
Для сборки программы в один независимый `.exe` файл убедитесь, что в папке со скриптом лежит файл `icon.ico` и запустите файл `build.bat`. 
Скрипт автоматически:
1. Установит необходимые зависимости из `requirements.txt`.
2. Скомпилирует готовый исполняемый файл в автоматически созданную папку `dist`.
3. Удалит временные файлы сборки (`.spec` и папку `build`).

## Использование
1. **Выбор источника:** Запустите программу. По умолчанию выбрана папка, где находится приложение. Вы можете нажать «Выбрать папку», чтобы обработать другую директорию, или «Выбрать файлы» для конвертации конкретных изображений.
2. **Папка сохранения:** По умолчанию результаты сохраняются в новую подпапку `output` рядом с оригиналами. При необходимости вы можете выбрать любую другую папку экспорта.
3. **Настройки:** Выберите быстрый режим на вкладке «Пресеты» или перейдите во вкладку «Пользовательские», чтобы детально настроить разрешение (с сохранением пропорций, обрезкой или рамками), качество и Lossless-сжатие.
4. **Запуск:** Нажмите «Начать обработку». Вы сможете отслеживать прогресс на полосе загрузки или прервать процесс кнопкой «Отмена». По завершении конвертации папка с готовыми `.webp` файлами откроется автоматически.
